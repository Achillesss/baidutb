# 百度贴吧小助手

- 自动签到：每天在设置的时间，将登记的用户所有贴吧进行签到

- 自动回帖：每天在设置的时间，使用配置好的文字回复已登记用户的所有贴吧的热贴

    目前只能使用统一文字对所有贴吧进行回复，没有开发在不同的时间对不同用户不同贴吧设置不同的回复文字。只因为现在够用，个人对这个功能并没有太大的需求。不排除后续会开发这个功能
